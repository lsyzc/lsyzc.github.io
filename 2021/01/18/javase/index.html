final关键字

1.修饰类：被修饰的类不能再有子类

2.修饰成员方法：此方法不能再被重写

3.修饰局部变量:

  • 基本数据类型：不可变指的是变量中的数据不可变
  • 引用数据类型：不可变指的是变量中的地址不可变，内容可变

4.修饰成员变量：

  • 由于成员变量具有默认值，因此用final修饰必须赋予初始值，要么直接赋值要么通过构造方法赋值。
  • 如果通过构造方法赋值，那么所有的重载构造方法都要对final修饰的成员变量赋值

hashSet集合

  • 底层采用数组+链表/红黑树的数据结构

  • 保证元素不重复的原理：

    • 比较两个待插入元素的哈希值
    • 不存在则插入到数组中去，存在再用equals方法比较是否相等，如果相等则不插入

  • 为自定义类型使用hashSet集合需要重写hashCode方法和equals和toString方法

  • package day04;

import java.util.HashSet;
import java.util.Objects;

public class Person {
    @Override
    public String toString() {
        return "Person{" +
                "age=" + age +
                ", name='" + name + '\'' +
                '}';
    }

    int age;
    String name;

    public Person(int age, String name) {
        this.age = age;
        this.name = name;
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof Person)) return false;
        Person person = (Person) o;
        return getAge() == person.getAge() && getName().equals(person.getName());
    }

    @Override
    public int hashCode() {
        return Objects.hash(getAge(), getName());
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public static void main(String[] args) {
        Person p1= new Person(20,"小明");
        Person p2=new Person(20,"小明");
        Person p3=new Person(30,"小红");
        HashSet<Person> set= new HashSet<>();
        set.add(p1);
        set.add(p2);
        set.add(p3);
        System.out.println(set);

    }
}
//[Person{age=20, name='小明'}, Person{age=30, name='小红'}]
