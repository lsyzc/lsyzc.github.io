linux基本命令：

一、文件目录类相关指令

pwd（当前路径）

cd

mkdir(默认只能创建一级目录)

​ -p（创建多级目录）

rmdir(不能删除非空目录)

​ 删除非空目录：rm -rf

touch 文件名称（创建新文件，可以同时创建多个文件）

​ touch hello.txt

cp

​ cp [选项] source dest

​ cp -r （递归复制整个文件夹）

​ 强制覆盖不提示 /cp

rm（移除文件或目录）

​ rm -rf(删除一个目录)

​ -r 删除整个文件夹

​ -f 强制删除不提醒

mv（移动文件与目录或重命名）

mv oldname newname（重命名）

cat（只读方式查看文件）

​ cat 选项 文件名

​ -n 带行号

​ cat /etc/profile.txt | more (分页展示)

​

less 指令（查看大文件，不是一次将大文件加载后才显示）

​

‘>’指令和’>>’指令(输出重定向，追加)

​ ls-l >a.txt(ls-l显示的内容覆盖写入到a.txt)

​ ls -l >>b.txt(ls-l显示的内容 追加到b.txt)

​ echo “内容” >/>> 文件

​ cal显示日历

echo指令

​ echo输出内容到控制台

​ echo 选项 输入内容

​ 使用echo输入环境变量，输出当前的环境路径

​ echo $path

head指令（显示文件开头部分的内容）

​ head 文件（查看文件头10行内容）

​ head -n 5 文件（查看头5行）

tail指令

​ 查看文件后10行

​ -n

​ tail -f文件（实时追踪文档的更新）

ln 指令(软链接创建快捷方式)

​ ln -s[原文件或目录] [软链接名]

​ 删除软链接 rm -rf 软连接名

​ 使用pwd命令时还是当前目录

history 指令（显示执行过的历史指令）

​ history 10 显示最近的十个

​ 执行历史指令 !编号

二、时间日期相关指令

date指令

​ date

​ date+%Y

​ date+%m

​ date+%d

​ H M S时分秒

​ eg: date “+%Y-%m-%d” 2020-12-25

​ 设置日期：

​ date -s “2018-2-2 11:23:4”

cal指令

​ cal 显示当前日历

​ cal 2020 显示2020整年

三、搜索查找指令

find指令

​ find 范围 选项

​ 按名查找 find /home -name hello.txt

​ 按文件的拥有者查找 -usr

​ 查找按大小 -size +20m

​ 大于：+ 小于：- 等于：直接写

​ eg: find -size +20M(大写)

locate指令

​ locate搜索文件 第一次运行前更新数据库 updatedb

grep指令

​ 过滤查找，管道符 | 将前一个命令的处理结果传递给后一个命令处理

​ 基本语法：grep [选项] 查找内容 源文件

​ -n 显示匹配行及符号

​ -i 忽略字母大小写

​ eg: 请在hello.txt文件中，查找“yes”所在行，并显示行号

​ cat hello.txt | grep -n yes

四、解压缩命令

gzip 和gunzip

​ eg: gzip hello.txt -> hello.txt.gz

​ 使用gzip压缩后不会保留原来的文件

zip 和 unzip

​ zip [选项] xxx.zip 将要压缩的内容

​ -r 递归压缩，即压缩目录

​ unzip：

​ -d<目录> 指定解压后文件的存放目录

​ eg: 将/home下的所有压缩成mypackage.zip

tar指令

​

​ example: tar -zcvf a.tar.gz a1.txt a2.txt（把a1.txt 和a2.txt压缩为a.tar.gz）

​ tar -zcvf myhom.tar.gz /home （将/home文件夹整体打包）

​ tar -zxvg a.tar.gz（解压到当前目录）

​ tar -zxvg a.tar.gz -C /opt（解压到opt目录 存在的目录）

linux实操

组

  1. 组的创建

    1. groupadd police（创建police组）
    2. useradd -g police tom（创建tom用户到police 组）

  2. 查看文件/目录所在组

    ls -ahl

  3. 修改文件所在的组

    • chgrp 组名 文件名

      应用实例：

      使用root用户创建文件 orange.txt，看看当前这个文件属于哪个组，然后将这个文件所在组修改到police组。

      chgrp police orange.txt

  4. 修改文件的所有者

    • chown 用户名 文件名

  5. 改变用户所在组（用root的管理权限）

    在添加用户时，可以指定该用户添加到哪个组中，同样用root的管理权限可以改变某个用户所在的组。
    改变用户所在的组：

    • usermod -g 组名 用户名

    • usrmod -d 目录名 用户名 改变用户登录的初始目录

      //example

      • 创建一个bandit组，将在police组的tom用户更改到bandit组。

        ​ groupadd bandit

        ​ usermod -g bandit tom

权限

rwx权限详解

权限的管理

通过chmod指令可以修改文件或目录的权限

  1. 通过+,-,=修改

    u->所有者 g->所在组 o->其他组 a->所有（u+g+o）

    w->写 r->读 x->可执行

    example:

    • chmod u=rxw,g=rx,o=x 文件或目录名
    • chmod o+w 文件或目录名
    • chmod a-x 文件或目录名

  2. 通过数字变更权限

    r=4 w=2 x=1

    usr group other

    chmod 771 file

    修改文件的所有者

chown newowner file

chown newowner:newgroup file

-R递归修改文件夹下所有文件的所有者

chown -R tom kkk/

    usermod -g police xh

  • jack更改组的权限为rw
